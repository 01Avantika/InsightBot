# InsightBot
Development of an AI-Powered Intelligent Data Analysis and Insight Generation System using LLMs and LangChain
It is a Streamlit-based web application that allows users to upload data files and instantly generate insights, visualizations, and executive-level reports using AI.  
The application focuses on simplicity, clean UI, and fast exploratory data analysis (EDA), making it ideal for beginners, analysts, and decision-makers.
